All scenes are children of the scene "Game Scene".

Collision layers:
	1: Environment
	2: Player Collision
	3: Enemy Collision
	4: Player hitboxes (a hitbox receieves damage, whereas a hurtbox deals damage)
	5: Enemy hitboxes

Ctrl + E = "Toggle Editable Children".

Trigger syntax is as follows: "NodeName,FunctionToCall,Delay(InDeciseconds)".
To add a trigger to an object, follow these steps:
	1. Create a string array for your condition and set it as an exported property.
	2. Create an instance of the TriggerHandler class and set the instace to be a child of the object.
	3. Whenever the condition is satisfied, use the Run function.
